Aplikasi ini adalah alat bantu konversi format file. Kami tidak menyimpan, mengindeks, atau menyediakan konten berhak cipta di server kami. Pengguna bertanggung jawab penuh atas file yang dikonversi. Pastikan Anda memiliki izin atau hak legal atas video yang Anda ubah formatnya.

3. Keamanan & Teknologi

Layanan ini menggunakan teknologi Client-Side WebAssembly. Artinya:

  • File Anda TIDAK PERNAH meninggalkan perangkat Anda.
  • Tidak ada risiko kebocoran data ke server cloud (karena kami tidak menggunakan server penyimpanan).
  • Proses konversi sepenuhnya bergantung pada spesifikasi perangkat (RAM/CPU) yang Anda gunakan.
