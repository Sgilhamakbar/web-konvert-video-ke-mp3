Kebijakan Privasi

Terakhir Diperbarui: Januari 2026

Terima kasih telah menggunakan Converter Video to MP3 Pro. Kami sangat menghargai privasi Anda. Dokumen ini menjelaskan bagaimana kami menangani data Anda dengan transparansi penuh.

1. Tidak Ada Pengumpulan Data (No Server Upload)

Ini adalah poin terpenting: Aplikasi ini beroperasi 100% secara lokal di perangkat Anda.

Saat Anda memilih file video untuk dikonversi:

  • File tersebut TIDAK diunggah ke server kami.
  • File tersebut TIDAK dikirim ke cloud atau pihak ketiga.
  • Proses konversi terjadi di dalam memori browser (RAM) laptop atau smartphone Anda menggunakan teknologi WebAssembly.

Setelah Anda menutup tab browser, semua data sementara akan hilang secara otomatis.

2. Izin Perangkat

Aplikasi ini mungkin meminta izin untuk membaca file penyimpanan Anda. Izin ini hanya digunakan untuk:

  • Mengambil file video yang Anda pilih secara spesifik untuk diproses.
  • Menyimpan file MP3 hasil konversi kembali ke folder Download perangkat Anda.

Kami tidak memiliki akses untuk membaca file lain selain yang Anda pilih secara manual.

3. Cookie dan Iklan (Google AdSense)

Agar layanan ini tetap gratis, kami bekerja sama dengan mitra periklanan. Berikut rincian penggunaan cookie:

  • Cookie Log & Preferensi: Kami menggunakan penyimpanan lokal (Local Storage) hanya untuk mengingat preferensi tema Anda (Mode Gelap/Terang).
  • Cookie Iklan Pihak Ketiga: Google, sebagai vendor pihak ketiga, menggunakan cookie untuk menayangkan iklan di situs ini. Google menggunakan cookie DART yang memungkinkannya menampilkan iklan kepada pengguna berdasarkan kunjungan mereka ke situs ini dan situs lain di Internet.

Anda dapat memilih untuk tidak menggunakan cookie DART dengan mengunjungi Kebijakan Privasi jaringan iklan dan konten Google.

4. Analitik & Keamanan

Aplikasi ini di-hosting menggunakan layanan penyedia statis. Penyedia hosting mungkin mencatat log akses standar (seperti Alamat IP, User Agent, dan waktu akses) untuk keperluan keamanan teknis dan pencegahan serangan DDoS. Log ini bersifat teknis dan tidak berisi konten file video Anda.
